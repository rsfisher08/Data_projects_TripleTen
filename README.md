# Data_projects_TripleTen
Projects I worked on while attending the Data Analysis TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | E-commerce Conversion Funnel, Cohort, and Retention Analysis| Transformed raw e-commerce event logs into actionable insights by building a user conversion funnel and cohort-based retention analysis in Google Sheets—calculating conversion and retention rates, identifying behavioral trends, and summarizing key business metrics for executive review.|
| 2 | [Saving Superstore: Profitability Analysis](https://github.com/rsfisher08/Data_projects_TripleTen/blob/main/SuperStore%20Returns%20RFisher.twbx) | Conducted a Tableau analysis of Superstore’s operations to identify top profit and loss drivers, evaluate advertising opportunities by region and season, and assess product return trends—delivering data-driven recommendations to improve profitability and reduce risk of bankruptcy.|
| 3 | SuperStore Return Analysis | Analyzed Superstore sales and returns data in Tableau to identify key drivers of product returns, built interactive dashboards to visualize trends by category, geography, and seasonality, and presented insights and recommendations to reduce return rates. |
| 4 | Shopify App Analysis | Analyzed publicly available Shopify App Store data in Power BI to identify key factors influencing app success through visualizations of app trends, review patterns, and developer responsiveness.|
